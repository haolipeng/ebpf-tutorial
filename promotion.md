## 🚀 从零开始学 eBPF：12 节课带你掌握 Linux 内核编程

### 你是否遇到过这些问题？

- 🤔 生产环境性能瓶颈，不知从何查起？
- 😰 SSL/TLS 加密流量，无法分析问题？
- 🔍 系统调用追踪，传统工具开销太大？
- 🌐 网络流量控制，想实现却无从下手？

**eBPF** 正是解决这些问题的终极武器！无需修改内核、无需重启系统，就能实现强大的监控、追踪和控制功能。

---

### 📚 一套完整的实战课程体系

这不是一个孤立的教程，而是一个**循序渐进、由浅入深的完整学习路径**：

#### 🎯 **第一阶段：入门三剑客（1-3 课）**

**Lesson 1: Hello World - 你的第一个 eBPF 程序**
- 搭建工程化开发环境
- 理解内核态与用户态分离的架构
- 5 分钟运行第一个程序

**Lesson 2: Kprobe - 追踪内核函数**
- 动态追踪任意内核函数（如 `do_sys_open`）
- 无需修改内核代码
- 实时监控文件操作行为

**Lesson 3: Uprobe - 追踪用户态函数**
- 追踪应用程序的任意函数
- 无需重新编译程序
- 监控 SSL/TLS 库函数调用

**学完你能做什么？**
✅ 追踪系统调用，监控进程行为
✅ 分析文件 I/O 性能问题
✅ 监控应用程序函数执行时间

---

#### 🔄 **第二阶段：数据通信（4-7 课）**

**Lesson 4: 用户态 Map 操作**
- 掌握 BPF Map 的增删改查
- 理解哈希表、数组等数据结构
- 实现统计计数功能

**Lesson 5: 内核态与用户态 Map 通信**
- 内核采集数据，用户态分析
- 实时数据传递机制
- 构建监控仪表盘基础

**Lesson 6: eBPF Go 语言开发**
- **Cilium ebpf-go**：纯 Go 语言开发
- **Aqua libbpfgo**：Go 封装 libbpf
- 两种方案对比，选择最适合你的

**Lesson 7: Ring Buffer & Perf Buffer**
- 高性能事件传递机制
- 适合大量数据采集场景
- 实现实时日志收集系统

**学完你能做什么？**
✅ 构建实时性能监控系统
✅ 实现进程行为审计工具
✅ 用 Go 语言开发 eBPF 应用

---

#### 🔬 **第三阶段：高级追踪（8-10 课）**

**Lesson 8: Tracepoint - 稳定的追踪点**
- 使用内核预定义的追踪点
- 追踪调度器、内存管理等子系统
- 接口稳定，跨内核版本兼容

**Lesson 9: Raw Tracepoint - 更底层的访问**
- 零开销的原始追踪点
- 访问内核原始数据结构
- 性能极致优化

**Lesson 10: BTF & CO-RE - 一次编译到处运行**
- 理解 BPF Type Format
- 实现跨内核版本的可移植性
- 编写一次，在不同内核上运行

**学完你能做什么？**
✅ 深入分析内核调度行为
✅ 编写可移植的 eBPF 工具
✅ 理解 Cilium、Falco 等工具原理

---

#### 🌐 **第四阶段：网络流量控制（11-12 课）**

**Lesson 11: TC Ingress - 入站流量控制**
- 在数据包进入网卡后处理
- 实现入站防火墙
- 过滤恶意流量、DDoS 防护

**Lesson 12: TC Egress - 出站流量控制**
- 在数据包离开网卡前处理
- 实现流量整形、速率限制
- 数据泄露防护（DLP）

**学完你能做什么？**
✅ 实现协议级防火墙（丢弃 ICMP、限制端口）
✅ 流量监控和统计分析
✅ 容器网络策略控制

---

#### 🔥 **第五阶段：生产实战（13 课）**

**Lesson 13: SSL/TLS 明文捕获**
- 使用 Uprobe 追踪 OpenSSL 函数
- 捕获加密前的明文数据
- 无需证书即可分析 HTTPS 流量

**真实场景应用：**
- 排查微服务间通信问题
- 分析 API 调用耗时
- 审计敏感数据传输
